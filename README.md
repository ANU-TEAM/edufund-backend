## Running the Project

Use the instructions below to run the project locally:

1. ```git clone https://github.com/ANU-TEAM/edufund-backend.git```
2. ```cd edufund-backend/```
3. ```composer install```
4. ```cp .env.example .env```
5. ```php artisan key:generate```
6. ```php artisan serve```
